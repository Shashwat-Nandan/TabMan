This tool generates icons for the TabMan extension. Click the download buttons below to save each icon.

Instructions:

  1. Click each "Download" button above to save the icon files
  2. Move the downloaded files to the icons/ folder in your TabMan extension directory
  3. Make sure the files are named exactly: icon16.png, icon48.png, and icon128.png
  4. Load the extension in Chrome by going to chrome://extensions/
  5. Enable "Developer mode" and click "Load unpacked"
  6. Select the TabMan folder
